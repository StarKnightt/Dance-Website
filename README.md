## Usage

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/yourusername/dance-academy-project.git

   ```
2. **Navigate to Project Directory:**
 ```bash
      cd DANCE-WEBSITE
 ```
3. **Install Dependencies:**
```bash
     npm install
```
4. **Set up MongoDB:**
- Ensure MongoDB is installed and running
- Update the connection string in `app.js` with your MongoDB URL : `await mongoose.connect("mongodb://localhost/contactDance");`

5. **Set The Project**
```bash
 npm start
```
6. Open in Browser

   Visit `http://localhost:80` to view the practice project.
